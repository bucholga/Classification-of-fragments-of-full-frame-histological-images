#LBL1 Валидация модели на части обучающей выборки
#LBL2 Автоматическое сохранение модели при обучении
#LBL3 Вывод различных показателей в процессе обучения (например, значение функции потерь на каждой эпохе)
#LBL4 Построение графиков, визуализирующих процесс обучения (график зависимости функции потерь от номера эпохи обучения, и т.п.)
#LBL5 Автоматическое тестирование на тестовом наборе/наборах данных после каждой эпохи обучения (при использовании итеративного обучения)
#LBL6 Использование аугментации и других способов синтетического расширения набора данных (дополнительным плюсом будет обоснование необходимости и обоснование выбора конкретных типов аугментации)
#LBL7 Возможность представления датасета в виде генератора. Исходный датасет имеет тип uint8, для работы с ImageDataGenerator необходимо преобразование во float, что приводит к увелечению занимаемого места на диске, из-за этого данные пришлось загружать частями.
